`wp theme mod remove` - Remove theme mod(s).

### OPTIONS

[&lt;mod&gt;...]
: One or more mods to remove.

[\--all]
: Remove all theme mods

### EXAMPLES

    wp theme mod remove --all
    wp theme mod remove background_color
    wp theme mod remove background_color header_textcolor
