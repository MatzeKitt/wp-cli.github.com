`wp role exists` - Check if a role exists.

<hr />

##DESCRIPTION

Will exit with status 0 if the role exists, 1 if it does not.

### OPTIONS

&lt;role-key&gt;
: The internal name of the role.

### EXAMPLES

    wp role exists editor
