`wp menu location list` - List locations for the current theme.

[\--format=&lt;format&gt;]
: Accepted values: table, csv, json, count, ids. Default: table

### AVAILABLE FIELDS

These fields will be displayed by default for each location:

* name
* description

### EXAMPLES

    wp menu location list
