`wp menu list` - Get a list of menus.

### OPTIONS

[\--fields=&lt;fields&gt;]
: Limit the output to specific object fields. Defaults to term_id,name,slug,count

[\--format=&lt;format&gt;]
: Accepted values: table, csv, json, count, ids. Default: table

### EXAMPLES

    wp menu list
