`wp term meta delete` - Delete a meta field.

<hr />

&lt;id&gt;
: The ID of the object.

[&lt;key&gt;]
: The name of the meta field to delete.

[&lt;value&gt;]
: The value to delete. If omitted, all rows with key will deleted.

[\--all]
: Delete all meta for the object.
