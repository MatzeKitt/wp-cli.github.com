`wp user remove-cap` - Remove a user's capability.

<hr />

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

&lt;cap&gt;
: The capability to be removed.

### EXAMPLES

    wp user remove-cap bob edit_themes
    wp user remove-cap 11 publish_newsletters
