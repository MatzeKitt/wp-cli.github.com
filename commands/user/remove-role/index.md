`wp user remove-role` - Remove a user's role.

<hr />

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

[&lt;role&gt;]
: A specific role to remove.

### EXAMPLES

    wp user remove-role bob
    wp user remove-role 12 editor
