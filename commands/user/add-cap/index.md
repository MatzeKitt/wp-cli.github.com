`wp user add-cap` - Add a capability for a user.

<hr />

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

&lt;cap&gt;
: The capability to add.

### EXAMPLES

    wp user add-cap john create_premium_item
    wp user add-cap 15 edit_product
