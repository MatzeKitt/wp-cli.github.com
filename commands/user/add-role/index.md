`wp user add-role` - Add a role for a user.

<hr />

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

&lt;role&gt;
: Add the specified role to the user.

### EXAMPLES

    wp user add-role bob author
    wp user add-role 12 author
