`wp user meta get` - Get meta field value.

### OPTIONS

&lt;user&gt;
: The user login, user email, or user ID of the user to get metadata for.

&lt;key&gt;
: The metadata key.

[\--format=&lt;format&gt;]
: Accepted values: table, json. Default: table

      Set the WordPress user
