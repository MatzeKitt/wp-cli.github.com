`wp user meta get` - Get meta field value.

<hr />

### OPTIONS

&lt;user&gt;
: The user login, user email, or user ID of the user to get metadata for.

&lt;key&gt;
: The metadata key.

[\--format=&lt;format&gt;]
: Accepted values: table, json, yaml. Default: table
