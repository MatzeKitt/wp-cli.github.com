`wp user meta add` - Add a meta field.

<hr />

&lt;user&gt;
: The user login, user email, or user ID of the user to add metadata for.

&lt;key&gt;
: The metadata key.

&lt;value&gt;
: The new metadata value.

[\--format=&lt;format&gt;]
: The serialization format for the value. Default is plaintext.
