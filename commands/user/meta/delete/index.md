`wp user meta delete` - Delete a meta field.

<hr />

&lt;user&gt;
: The user login, user email, or user ID of the user to delete metadata from.

&lt;key&gt;
: The metadata key.

[&lt;value&gt;]
: The value to delete. If omitted, all rows with key will deleted.
