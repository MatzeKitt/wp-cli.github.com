`wp plugin deactivate` - Deactivate a plugin.

### OPTIONS

[&lt;plugin&gt;...]
: One or more plugins to deactivate.

[\--all]
: If set, all plugins will be deactivated.

[\--network]
: If set, the plugin will be deactivated for the entire multisite network.
