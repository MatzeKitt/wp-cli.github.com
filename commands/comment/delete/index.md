`wp comment delete` - Delete a comment.

<hr />

### OPTIONS

&lt;id&gt;...
: One or more IDs of comments to delete.

[\--force]
: Skip the trash bin.

### EXAMPLES

    wp comment delete 1337 --force

    wp comment delete 1337 2341 --force
