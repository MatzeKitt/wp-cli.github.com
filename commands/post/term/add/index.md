`wp post term add` - Add a term. Appends to existing set of terms on the object.

<hr />

&lt;id&gt;
: The ID of the object.

&lt;taxonomy&gt;
: The name of the taxonomy type to be added.

&lt;term&gt;...
: The slug of the term or terms to be added.
