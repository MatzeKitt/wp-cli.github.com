`wp post term set` - Set terms. Replaces existing terms on the object.

<hr />

&lt;id&gt;
: The ID of the object.

&lt;taxonomy&gt;
: The name of the taxonomy type to be updated.

&lt;term&gt;...
: The slug of the term or terms to be updated.
